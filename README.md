# 💰 Expense Tracker App using Data Science

## 📌 Overview
This project is a data-driven Expense Tracker App that helps users analyze their spending patterns, visualize expenses, and make better financial decisions.

It simulates real-world financial analytics systems used in FinTech applications like Google Pay, CRED, and Mint.

---

## ❗ Problem
Managing expenses manually makes it difficult to:
- Understand spending patterns  
- Track monthly trends  
- Control overspending  

---

## ✅ Solution
This app:
- Tracks expenses  
- Analyzes spending patterns  
- Visualizes data using charts  
- Detects overspending  
- Provides an interactive dashboard  

---

## ✨ Features
- 📊 Category-wise expense analysis  
- 📈 Monthly trends  
- 🥧 Expense distribution  
- 💳 Payment mode analysis  
- 🚨 Overspending detection  
- 💻 Interactive Streamlit dashboard  

---

## 🛠️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Streamlit
